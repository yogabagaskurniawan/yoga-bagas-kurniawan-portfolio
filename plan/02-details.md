# 02 — Project Details (RTCC-O Framework)

## Role
AI berperan sebagai **Senior Frontend Developer & UI Designer** yang berpengalaman membangun portfolio website profesional dengan standar aksesibilitas dan performa tinggi.

## Task
Buat portfolio website single-page yang terdiri dari:
1. Header navigasi fixed + mobile hamburger menu
2. Section Hero dengan nama, role, CTA buttons, dan avatar animasi
3. Section About dengan bio dan stats card
4. Section Skills dengan progress bar tiap skill
5. Section Projects dengan project card grid
6. Section Contact dengan info kontak + form
7. Footer

## Context
- **Tech stack:** HTML5 semantik, CSS3 (custom properties, flexbox, grid), JavaScript vanilla
- **Target browser:** Chrome, Firefox, Safari modern
- **Deployed ke:** GitHub Pages (static hosting, tidak butuh backend)
- **Inspirasi design:** Dark luxury portfolio, editorial typography

## Constraints
- ✅ WAJIB menggunakan semantic HTML5: `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`, `<article>`
- ✅ WAJIB responsive mobile-first (breakpoint 640px dan 900px)
- ✅ WAJIB accessibility: `aria-label`, `aria-labelledby`, `role` attribute
- ❌ TIDAK BOLEH menggunakan framework CSS (Bootstrap, Tailwind)
- ❌ TIDAK BOLEH menggunakan framework JS (React, Vue)
- ❌ TIDAK BOLEH ada external dependency selain Google Fonts

## Output
- `index.html` — single file, semantic, bersih
- `style.css` — mobile-first, menggunakan CSS custom properties
- `script.js` — interaktivitas: nav toggle, scroll animation, form submit
- Semua kode dapat dijelaskan baris per baris ke recruiter
