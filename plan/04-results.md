## Reflection

**Yang berjalan baik:**
- Animasi IntersectionObserver membuat halaman terasa hidup tanpa library besar
- CSS custom properties sangat membantu konsistensi warna dan spacing
- Mobile-first approach memudahkan scaling ke desktop

**Yang bisa ditingkatkan:**
- Tambahkan dark/light mode toggle
- Hubungkan form ke layanan seperti Formspree agar email benar-benar terkirim
- Tambah lebih banyak proyek dengan filter kategori

**Pelajaran:**
- Semantic HTML bukan hanya soal SEO, tapi juga aksesibilitas screen reader
- `IntersectionObserver` jauh lebih efisien daripada event listener `scroll`
- Perencanaan di `plan/` membuat eksekusi lebih terarah dan terstruktur
