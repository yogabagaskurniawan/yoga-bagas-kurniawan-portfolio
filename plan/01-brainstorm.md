# 01 — Brainstorm

## Tujuan
Membuat portfolio website pribadi yang menampilkan identitas, keahlian, dan proyek saya sebagai Frontend Developer.

## Brainstorm dengan AI (Pertanyaan & Jawaban)

**Q: Apa saja section yang wajib ada di portfolio?**
A: Hero (perkenalan), About (tentang diri), Skills (keahlian teknis), Projects (karya), Contact (kontak).

**Q: Siapa target audience portfolio ini?**
A: Recruiter, klien freelance, dan sesama developer yang ingin berkolaborasi.

**Q: Tone dan visual seperti apa yang sesuai?**
A: Profesional tapi tetap personal. Dark theme dengan accent gold untuk kesan premium.

**Q: Tech stack apa yang digunakan?**
A: HTML5, CSS3 vanilla (mobile-first), JavaScript vanilla. Tidak menggunakan framework agar bisa di-deploy langsung ke GitHub Pages.

## Keputusan Desain
- **Warna:** Dark background (#0d0d0d) + gold accent (#e8c97a) → kesan premium
- **Font:** Playfair Display (display/heading) + DM Sans (body) → elegan + readable
- **Layout:** Mobile-first, grid responsive
- **Animasi:** Fade-in on scroll, skill bar animation, rotating avatar ring

## Target Deliverable
- File: `index.html`, `style.css`, `script.js`
- Deployed ke GitHub Pages
- Responsive di mobile dan desktop
