# Yoga Bagas Kurniawan — Portfolio Website

Portfolio website pribadi yang dibuat sebagai tugas **KAreerr Elevation Vol. 2**.

## 🔗 Live Demo
[\[GitHub Pages URL\]](https://yogabagaskurniawan.github.io/yoga-bagas-kurniawan-portfolio/)

## 📁 Struktur Repo

```
├── README.md
├── index.html          # Halaman utama portfolio
├── style.css           # Stylesheet (mobile-first, CSS custom properties)
├── script.js           # Interaktivitas (nav, scroll animation, form)
├── assets/
│   ├── desktop-view.png
│   └── mobile-view.png
└── plan/
    ├── 01-brainstorm.md
    ├── 02-details.md
    ├── 03-execution.md
    └── 04-results.md
```

## 🛠️ Tech Stack
- HTML5 Semantik
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript Vanilla (IntersectionObserver, Event Listeners)
- Google Fonts (Playfair Display + DM Sans)

## ✅ Fitur
- Responsive mobile-first (breakpoint 640px & 900px)
- Navigasi mobile dengan hamburger menu
- Animasi scroll menggunakan IntersectionObserver
- Skill progress bar dengan animasi
- Contact form dengan feedback interaktif
- Semantic HTML5 + ARIA attributes

## 📋 Cara Deploy ke GitHub Pages
1. Push semua file ke repository GitHub
2. Buka **Settings** → **Pages**
3. Source: `main` / `(root)` → **Save**
4. Tunggu beberapa menit → live di `https://yogabagaskurniawan.github.io/yoga-bagas-kurniawan-portfolio/`
